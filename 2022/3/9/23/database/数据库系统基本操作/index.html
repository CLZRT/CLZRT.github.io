    • 查看所有数据库: show databases;

选择数据库

  • 语句:
    • use 数据库名称;
    • 登陆时直接进入数据库
      • 密码显示
        • mysql -u 用户名 -p 密码 数据库名;
      • 密码隐藏
        • mysql -u 用户名 -p 数据库名;

删除数据库

  • 语句:
    • drop database mydb;(不存在即报错)
